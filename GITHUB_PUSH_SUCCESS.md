## ğŸ“‹ Setup Instructions

### **1. Clone Repository**
```bash
git clone https://github.com/Silent-bat/builder.git
cd builder
```

### **2. Install Dependencies**
```bash
npm install
# or
pnpm install
```

### **3. Setup Environment**
```bash
# Copy environment template
cp .env.example .env

# Configure database
DATABASE_URL="file:./prisma/dev.db"

# Add other required env vars (see .env.example)
```

### **4. Setup Database**
```bash
# Push schema to database
npx prisma db push

# (Optional) Seed with sample data
npx prisma db seed
```

### **5. Run Development Server**
```bash
npm run dev
```

### **6. Build for Production**
```bash
npm run build
npm start
```
